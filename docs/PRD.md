## Goal
사용자 **본인의 목소리를 AI로 복제**해, "내 목소리로 가족에게 사기 전화가 가면?"을 직접 체험시키는 AI 금융사기 백신을 만든다. 사용자가 30초 목소리를 녹음하면 AI가 그 목소리로 가족 사칭 사기 전화를 앱 내 오디오로 재현하고("이렇게 감쪽같이 당한다"를 체감), AI 사기범과의 실시간 역할극 후 "어디서·언제 속았는지"를 짚는 취약점 리포트와 방어 훈련(게임화)을 제공한다. 약화된 실전 공격에 미리 노출시키는 예방접종(inoculation)형 서비스다.

### 해결하려는 문제
- 기존 안티사기 서비스는 탐지·차단(사후) 중심이거나, 고정 시나리오 재연 교육에 그친다. 사람들은 사기의 존재를 지식으로는 알아도, 실제 상황의 심리적 압박(권위·긴급·공포·가족애)을 겪어본 적이 없어 막상 당한다.
- 격차는 "지식"이 아니라 "경험·자각"에 있다. 특히 **본인 목소리가 얼마나 쉽게 탈취·악용되는지**를 신체적으로 자각할 방법이 없다.

### 근거 데이터 (조사 기준일 2026-07-21)
- 부산 60대 딥보이스 가족사칭 실사례, 2천만원 송금 피해.
- 2026년 1~5월 보이스피싱 피해액 2,564억원(전년 동기 대비 +50%).
- 목소리 탈취가 "여보세요" 한마디로도 가능할 만큼 쉬움. 경찰청 딥페이크 "자녀 납치" 사기 주의보.
- 기존 예방 대책은 "가족 인증 키워드 정하기" 등 수동 교육뿐.

### 차별점 (참신점 3가지)
1. **본인 목소리 복제**로 위협을 신체적으로 자각(경쟁사는 고정 시나리오/타인 음성).
2. **가족 원격 훈련** — 자녀가 부모의 계정 생성 UI를 보조하고, 목소리 등록(녹음)은 반드시 부모 본인이 직접 수행(안전 기본값 확정, OQ-13 resolved → AC-020 정합). 부모가 안전하게 당해보고 리포트를 남긴다(리포트를 자녀가 열람 가능한지는 OQ-15 미확정). 자녀 보조 UI의 구체 권한 범위·MVP 실제 포함 여부는 OQ-3.
   - **발표 내러티브 강조 확정(2026-07-21), 단 Day1 빌드 범위는 미확장(본인 온보딩만).** 발표(피칭)에서는 이 "가족 원격 훈련"을 비중 있게 내세우기로 확정했으나, Day1 실제 빌드 범위는 지금 그대로 유지한다: 본인 온보딩만 구현하고, 자녀는 계정 생성 UI 보조 수준까지만(로드맵 방향성). **OQ-3(자녀 보조 UI의 MVP 실제 포함 여부)·OQ-15(자녀 리포트 열람 가능 여부)는 빌드 스코프 관점에서는 여전히 open이다.** 발표에서 강조한다는 방침과 빌드 스코프 결정은 별개 사안임(발표팀 가이드라인은 Risks 표 참조).
3. **게임화** — 사기 방어 등급, 신종 수법 자동 업데이트(구독 확장 방향).

경쟁사 대비: 국내 은행(우리 '하마터면', 신한 '보이스피싱제로')·해외 B2B(Brightside/Hoxhunt/Wizer/Doppel)·통신사앱(시티즌코난/익시오)은 고정 재연·기업 직원 대상·탐지 차단(사후)에 머문다. 우리는 개인·가족(어르신) B2C + 한국어 + 본인 목소리 감정 체험 + 사전 예방접종.

## Acceptance Criteria
| ID | Verifiable Condition |
|---|---|
| AC-001 | 시나리오 라이브러리는 최소 한국어 "가족 납치/사고(딥보이스)" 1종을 포함하고, 세션 시작 전 목록에서 선택할 수 있다(여유 시 "기관사칭(검찰/은행) 문자+링크" 1종 추가). |
| AC-002 | 각 시나리오는 시작 전 제목·사기 유형·예상 소요 시간·난이도(강도) 라벨을 표시한다. |
| AC-003 | 세션이 시작되면 AI는 선택된 시나리오의 사기범 인격으로 대화를 시작하고 세션 내내 그 인격을 유지한다(명시적 안전 종료 경로 제외하고 캐릭터를 깨지 않는다). |
| AC-004 | AI는 각 사용자 턴에 대해 정의된 지연 목표 내에 응답한다(목표: 95% 턴 기준 ≤10초, 정확한 값은 OQ-9). |
| AC-005 | AI의 수법은 사전 정의된 "약화된" 수법 집합에서만 나오며, 현실에서 사기를 실행하는 데 쓰일 수 있는 운영 정보(실제 계좌번호, 실제 송금 절차, 실제 악성 앱/링크)는 절대 제공하지 않는다. |
| AC-006 | 사용자는 모든 턴에서 항상 보이는 단일 컨트롤로 세션을 즉시 종료할 수 있고, 종료 시 세션이 끝나고 리포트로 진행한다. |
| AC-007 | 세션은 정의된 종료 조건(사용자 완료/속아 넘어감, 사용자 종료, 최대 턴·시간 한도 도달) 중 하나로 끝나며, 종료된 모든 세션은 정확히 1개의 리포트를 생성한다. |
| AC-008 | 세션 종료 후 리포트는 최소한 (1) 사용자가 속았거나 응했을 특정 메시지/순간, (2) 사용된 조작 수법, (3) 구체적 예방 조언 1개 이상을 식별한다. |
| AC-009 | 사용자가 한 번도 속지 않은 경우, 리포트는 이를 명시하고 그럼에도 시도된 수법들을 나열한다. |
| AC-010 | 세션 완료 시 세션 결과로부터 방어 점수/등급이 산정되어 사용자에게 표시된다(기본형, 정확한 산정식은 OQ-5). |
| AC-011 | 사용자의 등급/레벨과 세션 횟수는 세션 간 지속되며 프로필에서 확인할 수 있다. |
| AC-012 | 첫 세션 전(및 요청 시 언제든) 사용자는 해당 콘텐츠가 훈련용 시뮬레이션이며 실제 금전·자격증명이 관여하지 않는다는 점과 종료 방법을 안내받는다. |
| AC-013 | 사용자가 실제 사기 실행 방법을 요청하거나 AI를 타인 사기 학습에 이용하려 하면, 시스템은 거부하고 운영 가능한 사기 지침을 제공하지 않는다. |
| AC-014 | 정의된 최소 연령 미만 사용자의 접근을 제한하는 연령 확인이 존재한다(최소 연령 = **만 14세**, OQ-8 확정). |
| AC-015 | 세션 종료 시 사용자에게 상호작용이 시뮬레이션이었음을 상기시키는 디스컬레이션(안심) 메시지가 표시된다. |
| AC-016 | 사용자는 과거 세션과 리포트의 시간순 히스토리를 볼 수 있다(P1). ※ "계정 생성/접근(로그인·인증)" 부분은 P0로 상향되어 **AC-027**로 분리 관리한다. AC-016은 히스토리 열람 기능(P1)만을 가리킨다. |
| AC-017 | 최초 사용 시, 사용자는 어떤 시나리오도 시작되기 전에 시뮬레이션된 기만 경험에 대해 명시적으로 동의해야 하며, 동의 없이는 세션이 시작되지 않는다. |
| AC-018 | (음성 클론) 사용자가 앱에서 30초 분량의 지정 대본을 녹음하면, ElevenLabs Instant Voice Cloning으로 해당 목소리의 클론이 생성되고 임의 한국어 문장을 그 목소리로 합성해 재생할 수 있다. |
| AC-019 | (딥보이스 재생) 클론된 목소리로 "가족 납치/사고" 사칭 대사(예: "엄마 나 사고났어, 합의금 지금…")가 **앱 내 오디오로 재생**된다. 실제 전화망/통화 연동 없이 인앱 재생 전용이다. |
| AC-020 | (가드레일1·본인 동의) 사용자는 자신의 목소리만 등록할 수 있으며, 녹음/등록 전 본인 동의 및 본인 확인 절차를 통과해야 한다. 타인 목소리를 무단 등록·클론하는 경로가 UI에 존재하지 않는다. |
| AC-021 | (가드레일2·즉시 폐기) 훈련 세션 종료 시 생성된 클론 음성(및 기타 합성물) 파일이 스토리지에서 삭제되고 서버에 영구 저장되지 않으며, 삭제 이벤트가 로그로 남는다. |
| AC-022 | (가드레일3·합성 표식) 모든 생성물(재생되는 합성 음성·이미지)에 "AI 훈련용 합성" 표식(오디오 안내 문구 또는 화면 라벨)이 포함된다. |
| AC-023 | (가드레일4·송금 유도 없음) 시뮬레이션은 실제 계좌·송금·이체 기능을 포함하지 않으며, 세션 종료 시 "이것은 훈련이었습니다"를 명시적으로 고지한다. |
| AC-024 | (가드레일5·PII/인젝션) 대화 로그의 개인정보는 마스킹/토큰화되고, 사용자 입력과 시스템 프롬프트가 분리되어 프롬프트 인젝션으로 사기범 인격·가드레일이 해제되지 않는다. |
| AC-025 | (P1·사칭 이미지) 사칭 이미지 1종(예: 가짜 송금완료 화면)이 생성되어 시뮬레이션 중 표시되며, AC-022의 합성 표식을 포함한다. |
| AC-026 | 취약점 리포트는 사용자가 속은 시점을 시간축(예: "15초 시점에 속았습니다")으로 표시하고, 그 순간 취했어야 할 올바른 대처(끊고 직접 전화 등)를 제시한다. |
| AC-027 | (로그인·P0) 사용자는 온보딩 시작(사전 고지·명시적 동의) 전에 Firebase Auth로 계정을 생성하거나 로그인해야 하며, 인증되지 않은 사용자는 이후 단계(동의·목소리 녹음·세션·리포트)로 진행할 수 없다. 인증 성공 후 발급된 본인 계정(uid)에 이후 모든 데이터(동의 로그·리포트·메타)가 귀속된다. 구체 인증 프로바이더(이메일/구글 소셜/전화번호 등) 선택은 architect 소관(docs/UX.md OQ-U5). |

## Out of Scope (Future)
사용자가 하루 완성을 위해 명시적으로 제외한 항목.

- **영상 딥페이크(talking-photo, D-ID 등)** — 완전 제외. 로드맵으로만 언급(P2). 사용자 명시 지시.
- **실시간 음성 통화(전화망 연동)** — 제외 → 앱 내 오디오 재생(AC-019)으로 대체.
- **실제 STT(음성 인식 대화)** — 제외 → 텍스트 채팅 역할극. 음성은 클론 결과 재생 전용, 사용자 입력은 텍스트.
- **다국어·다수 시나리오** — 제외 → 한국어 "가족 납치" 1종 집중(여유 시 "기관사칭" 1종).
- **기업 관리자 대시보드 / 조직 단위 분석** — 이번 스코프 제외(B2B는 시장성 로드맵으로만).
- **AI Hub 딥페이크 변조 영상 데이터셋 코어 도입**(탐지 모델 학습) — 2일 내 학습 불가·승인/용량/반입 리스크. 코어 미도입, 데모용 샘플 전시 정도만 제한적 활용 가능.
- 로맨스스캠 시나리오(v0.1엔 포함, 하루 스코프서 제외).
- 리딩방·투자 딥페이크 영상 시나리오(영상 제외에 종속).
- 실시간 사람 개입(human-in-the-loop) 코칭.
- 통신사·은행 실사기 경보 시스템 연동.

## Assumptions
- MVP는 웹(Next.js) 단일 채널, 텍스트 채팅 기반이며 사용자는 인터넷 연결 기기에서 텍스트를 읽고 입력하고 마이크로 30초 녹음할 수 있다.
- ElevenLabs Instant Voice Cloning API가 1~3분(본 스코프는 30초) 샘플로 사용 가능한 품질의 한국어 클론을 하루 내 산출한다(전날 리허설로 검증 — Risk 참조).
- 지속적 인격 롤플레이와 가드레일을 감당할 LLM(Claude/Gemini)을 세션당 합리적 비용/지연으로 사용할 수 있다.
- 사용자는 자발적 훈련 대상이며 공격자가 아니다.
- **자녀 대리설정 시, 목소리 녹음은 반드시 본인(등록 대상자)이 직접 수행하며 자녀는 계정 생성 UI 조작만 보조한다. 실제 녹음 버튼은 본인이 누른다**(OQ-13 확정). 따라서 동의 주체는 본인, 데이터 소유권은 본인 계정 소유다(OQ-11 확정).
- 1차 언어·시장은 한국어/한국(확정, OQ-7 resolved).

## Constraints
- **기술 스택(확정)**: Next.js + Firebase(Auth/Functions/Firestore/Storage) + LLM(Claude/Gemini, 사기범 역할극) + ElevenLabs Instant Voice Cloning API($5/월, 1~3분 샘플로 클론).
- **팀 구성**: 개발 코어 3명(제윤·진우·수홍) 병렬 작업 / 기획·발표 2명(한솔·광훈).
- **절대 마감(가장 강한 제약)**: 실질 개발은 Day1 하루(무박 야간 포함)에 전부 완료. Day2 오전은 개발 신규 금지(버그픽스·데모 안정화·녹화만). 최종 마감 = Day2 오전 11시.
- **대회 배점**: 시장성 35% / 보안 20% / 기술·완성도·발표 45%. 보안 가드레일(AC-020~AC-024, AC-017, AC-015, AC-012)이 보안 배점의 핵심 근거.
- **법적/윤리 (필수)**: 현실 사기 운영 지침 미제공(AC-005, AC-013), 실제 금전 피해 방지(실계좌·실송금·실링크 금지, AC-023), 본인 목소리만 등록(AC-020), 생성물 즉시 폐기·미저장(AC-021), 합성 표식(AC-022), PII 마스킹·PIPA 준수(AC-024), 사전/사후 고지(AC-012, AC-015). 미성년자 보호(AC-014, P1).
- **예산**: ElevenLabs $5/월 확정. 총 예산·API 크레딧 상한액 수치는 미확정이나, **T1(ElevenLabs 클론 PoC) 착수 전 사용자가 직접 크레딧/rate limit을 확인한 뒤 진행**하기로 확인 절차는 확정(OQ-14).

## Risks
확정안 리스크 + v0.1 리스크 중 이번 스코프에서도 유효한 것만 통합.

| Risk | Impact | Mitigation |
|---|---|---|
| 윤리 Q&A 공격("악용되면?") — 발표 심사에서 정면 공격 예상 | High | 보안 가드레일 5종(AC-020~AC-024)으로 방어: 본인 목소리만·즉시 폐기·워터마크·송금 유도 없음·PII/인젝션 방어. 발표 시 명시. |
| 음성 클론 품질(한국어·감정·지연) 미달 | High | ElevenLabs 전날 리허설로 사전 테스트 필수. 실패 대비 사전 녹화본 확보. PoC를 Day1 오전 최우선 태스크로 선(先)수행해 리스크 조기 제거. |
| 영상 유혹으로 인한 스코프 크립 | High | 영상 딥페이크 완전 제외(Out of Scope), 로드맵으로만 언급. 하루 스코프 사수. |
